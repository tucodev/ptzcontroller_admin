

보안상의 이유로 브라우저에서 exe 파일을 자동으로 다운로드하고 실행하는 것은 불가능합니다. 

하지만 다음과 같은 방식으로 사용자 경험을 개선할 수 있습니다:


가능한 방식

1. 프록시 자동 감지 + 다운로드 안내
┌─────────────────────────────────────────────────────────────┐
│  웹 UI 로딩 시                                               │
│  ├─ 프록시 연결 시도 (ws://localhost:8765)                    │
│  │                                                          │
│  ├─ 연결 성공? → 정상 사용                                     │
│  │                                                          │
│  └─ 연결 실패? → 다운로드 안내 팝업 표시                         │
│       ├─ "프록시 다운로드" 버튼 클릭 → exe 다운로드              │
│       └─ 사용자가 실행 후 "재연결" 클릭                         │
└─────────────────────────────────────────────────────────────┘


2. Custom URL Protocol (선택사항)

프록시가 설치되면 ptz-proxy:// 프로토콜 등록
웹에서 ptz-proxy://start 클릭 시 프록시 자동 실행
단, 최초 1회는 사용자가 수동 설치해야 함

-------------------------------------------------------
구현 범위                    기능 가능 여부
-------------------------------------------------------
exe 파일 다운로드 링크 제공      ✅ 가능
프록시 실행 여부 자동 감지       ✅ 가능
프록시 미실행 시 안내 팝업       ✅ 가능
exe 자동 다운로드 (클릭 없이)    ❌ 브라우저 보안 정책으로 불가
exe 자동 실행     	          ❌ 브라우저 보안 정책으로 불가
-------------------------------------------------------

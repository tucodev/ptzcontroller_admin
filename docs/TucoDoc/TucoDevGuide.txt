1. 다음을 실행해서 db를 만든후 cloudtype 으로 배포할 것

yarn install
yarn prisma generate
yarn prisma migrate dev
yarn prisma db seed



2. DB reset
  yarn prisma migrate reset



3. 다음은 db 확인 툴을 돌리는 것이다.

npx prisma studio "[db connection url]"
or 
yarn prisma studio "[db connection url]"


[example]
npx prisma studio "postgresql://neondb_owner:npg_cP1qQeFoMkO3@ep-patient-waterfall-a1tk4pzw-pooler.ap-southeast-1.aws.neon.tech/neondb?sslmode=require&channel_binding=require"

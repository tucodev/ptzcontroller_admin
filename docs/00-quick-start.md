# PTZ Controller — Quick Start Guide

- 제공 : TYCHE Inc (주식회사 타이키)

> 웹 브라우저에서 PTZ Controller에 처음 접속하는 사용자를 위한 빠른 시작 가이드입니다.

---

## 1. 회원가입

1. 브라우저에서 PTZ Controller 주소에 접속합니다.
   ( https://ptz.tycheai.xyz )
2. 로그인 화면 하단의 **"Don't have an account? Sign up"** 을 클릭합니다.
3. 이름, 회사/소속, 이메일, 비밀번호(8자 이상)를 입력합니다.
4. **"Create Account"** 를 클릭하면 가입이 완료되고 자동으로 로그인됩니다.

---

## 2. 카메라 추가

1. 대시보드 왼쪽 **Cameras** 영역의 `+` 버튼을 클릭합니다.
2. 다음 항목을 입력합니다.

| 항목           | 설명                    | 예시                       |
| -------------- | ----------------------- | -------------------------- |
| Camera Name    | 카메라 이름             | 1번 카메라                 |
| Protocol       | 통신 프로토콜           | PelcoD / Ujin / ONVIF      |
| PTZ 카메라 IP  | 카메라 IP (Proxy 기준)  | 192.168.1.100              |
| Port           | 통신 포트               | 4001                       |
| Device Address | 장치 주소 (PelcoD/Ujin) | 1                          |
| Proxy URL      | PTZ Proxy 주소          | ws://localhost:9902 (유지) |

    (주의) Proxy URL 의 경우 기본 환경 사용시에는 변경하지 마십시요.

3. **Save** 를 클릭합니다.

> **ONVIF 카메라**: Profile Token은 비워두면 자동으로 조회됩니다.

---

## 3. PTZ Proxy 준비

PTZ Controller는 카메라에 직접 접근을 위하여 **PTZ Proxy**가 필요합니다.

```
브라우저 ←→ PTZ Proxy ←→ PTZ 카메라 (TCP), 시리얼 추후 지원 예정
```

- PTZ Proxy를 카메라에 접근 가능한 PC에서 실행합니다.
- 기본 포트: **9902**

> PTZ Proxy를 실행하지 없은 상태에서 연결을 시도하면 **다운로드 안내 팝업**이 자동으로 나타납니다. Cloud Download나 직접 실행파일 다운로드 버튼을 통해 설치 파일을 다운로드 하고 설치한 후 실행하세요.

---

## 4. 카메라 연결 및 제어

1. 카메라 목록에서 카메라를 클릭하여 선택합니다.
2. **Connect** 버튼을 클릭합니다.
3. 연결 상태가 **connected** 로 바뀌면 제어할 수 있습니다.

### 기본 제어

| 제어           | 조작                                             |
| -------------- | ------------------------------------------------ |
| **Pan / Tilt** | 방향 버튼을 **누르고 있는 동안** 이동, 떼면 정지 |
| **Zoom**       | In / Out 버튼                                    |
| **Focus**      | Near / Far 버튼                                  |
| **속도 조절**  | 각 슬라이더 (1–100%)                             |
| **프리셋**     | 1–6 빠른 버튼 또는 숫자 입력 후 Go / Set         |

---

## 5. Hex 모니터

화면 하단의 **Hex Monitor** 에서 카메라와 주고받는 패킷을 실시간으로 확인할 수 있습니다.

- **TX** (초록): 전송한 명령
- **RX** (파랑): 카메라 응답
- 상단 바에 Pause / Export / Clear 버튼 제공

> ONVIF는 HTTP 통신이므로 TX가 표시되지 않는 것은 정상입니다.

---

## 6. 설정

헤더의 ⚙️ 아이콘을 클릭하면 기본 설정을 변경할 수 있습니다.

| 설정                 | 설명                                 |
| -------------------- | ------------------------------------ |
| Default Protocol     | 새 카메라 추가 시 기본 프로토콜      |
| Proxy WebSocket Port | 기본 Proxy 포트                      |
| Theme                | Light / Dark / System (기본: System) |

---

## 7. 비밀번호를 잊었을 때

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 를 클릭합니다.
2. 가입한 이메일을 입력하면 재설정 링크가 발송됩니다.
3. 이메일의 링크를 클릭하여 새 비밀번호를 설정합니다.

